linux commands

ctrl + alt + T- команда открытия терминала

~ - символ домашней директории

/ - символ корневой директории

. - обазначает текущую директорию

.. - обазначет предудущую директорию

1.ls – команда для вывода всех директории и файлов в текущей 
директории в терминале

2. cd (change directory) – комната для перехода из одной 
директории в другую 

 3. mkdir – создаёт директории

4.  touch – создаёт файл

5. rmdir – удаляет пустую директорию

6. rm – удаляет файл

7. rm -rf – удаяляет вообще все (принудительное удаление)

8. ls -a – показывает скрытые и обычные файлы и директории
ls -la – показывает подробную инфу

9. pwd (print working directory) – выводит путь по текущей директории

10. cp -  копирует файл или директорию в указоное место
            cp <название> <куда>

11.mv – команда для перемещения файла или деректории, ф также для переименования файла или директории 
                переименование: mv john.txt jamie.txt
                 перемещение: mv john.txt directory/

12. cat – показывает содержание файла

13. nano -  открвает файл для редактирование в графическом интерфейсе НАНО *если нет файла, то НАНО создаст новый

14. echo – команда для быстрого записи текста в файл 
        echo текст  > файл
